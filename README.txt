git bash利用手順

▲▽▲▽▲▽▲▽▲▽▲▽初回△▼△▼△▼△▼△▼△▼△▼

1.gitがインストールされていない場合gitインストール(versionは最新版とかでいい)

2.好きな場所にディレクトリを作成(今後最新verを引っ張る為に利用するので使いやすいところに作る)
名前は	...\minecraft\1.10.2\
		     \1.7.2\
		     \.....
	のようにversion毎にフォルダを作成することを推奨

3.リポジトリから持ってくるバージョンを右クリック
	-> 1.10.2のファイルをダウンロードしたいなら1.10.2のフォルダを右クリック
GitBash Here　とあるはずなのでそれをクリックし、コンソールを出す

4.以下のコマンドを実行

$ git clone https://github.com/Smichroon/minecraft-1.10.2

すると指定したディレクトリ内にmodsなどのファイルがダウンロードできる（はず）

5.ダウンロードしたフォルダを.minecraftフォルダにすべて上書きする(ショートカット作成推奨)

参考パス：
...\ユーザー名\AppData\Roaming\.minecraft	//minecraftの本体フォルダパス

6.forgeをインストールする(ランチャーのある階層に)
基本的には自動で.minecraftの階層指定してくれてるし、そこでおっけ

以上ッ！閉廷ッッ！！！！！

▲▽▲▽▲▽▲▽▲▽▲▽初回以降△▼△▼△▼△▼△▼△▼△▼

差分があって管理者から更新依頼が来た際(新しいmodなどを導入した際)に最新版をダウンロードする手順

1.初回の手順で記載した1.10.2などのクローンしたフォルダを右クリック->GitBash Here

2.以下のコマンドを実行し、差分を読み込む

$ git fetch

3.以下のコマンドを実行し、差分をフォルダに更新する

$ git merge origin master

4.追加、削除、変更のあったファイルと変更行数が表示されて、更新が成功したら完了

5.ダウンロードしたフォルダを.minecraftフォルダにすべて上書きする(ショートカット作成推奨)

参考パス：
...\ユーザー名\AppData\Roaming\.minecraft


以上ッ！閉廷ッッ！！！！！
